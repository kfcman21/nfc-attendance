# NFC 학생 출석 프로그램

NFC 카드를 태그해 학생 출결을 기록·관리하는 **데스크톱 앱(Electron)** 입니다.
인터넷 없이 PC 한 대에서 동작하며, 데이터는 **SQLite**로 로컬에 저장됩니다.

- **`npm start`** 하면 데스크톱 창이 자동으로 뜸 (브라우저 따로 열 필요 없음)
- 카드를 대면 실시간으로 출석·점수가 기록되고 대시보드로 한눈에 확인
- 모든 자료를 **엑셀(CSV)** 로 내보내거나 **전체 백업(.db)** 가능
- (선택) **구글 시트 동기화**, **공휴일·미세먼지·날씨** 공공데이터 연동

> 구조: Electron 창이 화면을 표시하고, 리더기/DB는 내부적으로 작은 Node 서버
> (`src/server.mjs`)가 담당합니다. 사용자는 `npm start` 만 하면 됩니다.

## ✨ 주요 기능

| 기능 | 설명 |
|---|---|
| **번호별 등록** | 개인정보(이름) 없이 **출석번호만으로** 카드 등록 가능. "번호 빠른 등록"으로 카드만 연속 태그하면 번호 자동 증가 |
| **정시/지각 점수** | 정해진 **마감시각** 이전 태그 = 만점, 이후 = 지각(0점). 설정에서 시각·만점 조정 |
| **대시보드** | 출석/미출석·출석률·정시/지각·오늘 점수·최근 7일 추이·감정 분포 |
| **주간 점수** | 학생 × 요일 점수표, 주 이동, 엑셀 내보내기 |
| **데이터 관리** | 전체 백업(.db)·항목별 엑셀 저장·항목별/전체 초기화 |
| **두 가지 리더기** | 시리얼(CR-100)과 **HID(키보드형)** 리더기 모두 지원 |
| **구글 시트 동기화** | 출석을 구글 시트에 자동 기록 (선택) |
| **공공데이터 연동** | 공휴일 자동 반영, 미세먼지·날씨 위젯(체육 야외활동 판단) (선택) |
| **NEIS 학교 연동** | 오늘 급식·알레르기 표시, 학사일정(휴업일·방학)을 결석에서 자동 제외 (선택) |
| 그 외 | 감정 출석부·포인트 상점·도서 대여·셔틀런·서킷 트레이닝 |

## 🚀 실행 / 빌드

```
npm install         # 최초 1회
npm start           # 데스크톱 앱 실행 (창 자동 표시)

npm run dist        # 배포용 .exe (설치본 + 포터블) 생성 → dist/
npm run icon        # 로고(build/logo.svg)로 아이콘 다시 굽기
```

- 결과물(`dist/`): **`NFC학생출석-<버전>-설치.exe`**(설치형), **`NFC학생출석-<버전>-portable.exe`**(무설치)
- ✅ **완전 독립형** — 대상 PC에 Node.js 설치 불필요(Electron 내장 Node로 구동, serialport는 프리빌트)
- 데이터 저장 위치(배포 실행 시): **`%AppData%\nfc-attendance\attendance.db`** — 업데이트해도 보존
- 하드웨어 없이 시험: 화면 아래 **"테스트: 가짜 카드 태그"** 버튼

## 🖥 화면(탭)별 사용법

- **실시간 출석** — 카드를 대면 **① 출석·점수 → ② 오늘 기분(감정 체크) → ③ 「오늘은 이런 날!」(급식·학사일정·날씨·미세먼지)** 순으로 안내. (③은 켜둔 연동이 있을 때만 표시) 상단에 날씨·미세먼지·급식 위젯도 표시.
- **대시보드** — 그 날 출석/미출석·출석률·정시/지각·획득 점수·7일 추이.
- **📅 주간 점수** — 학생×요일 점수표(정시=만점, 지각/결석=0). 주 이동·**엑셀 내보내기**. 공휴일 기능을 켜면 주말·공휴일은 결석에서 제외.
- **학생 관리** — 카드 UID 칸을 클릭하고 카드를 태그하면 자동 입력. **번호만 입력해도 등록**(이름은 선택). "번호 빠른 등록"으로 일괄 등록.
- **출석 현황** — 날짜별 출석 조회 + CSV(엑셀) 내보내기(번호·상태·점수 포함).
- **💾 데이터 관리** — 전체 백업·항목별 엑셀 저장·초기화(아래 참고).
- **⚙ 설정·진단** — 점수 마감시각, HID 입력, 구글 시트, 공공데이터, 리더기 포트.

---

## ⏱ 출석 점수 (정시 / 지각)

- **설정·진단 → 출석 점수**에서 **마감시각**과 **만점**을 정합니다(기본 09:00 / 10점).
- 마감시각 **이전(포함)** 태그 → **정시(만점)**, 이후 → **지각(0점)**, 미태그 → 결석(0점).

---

## ⌨ HID(키보드형) 리더기

키보드처럼 카드 번호를 입력하고 Enter를 보내는 리더기를 쓸 때 사용합니다.
**설정·진단 → 키보드(HID) 리더기 입력**을 켜면, 입력칸 바깥에서 들어온 번호를 출석으로 처리합니다. (시리얼 CR-100과 동시 사용 가능)

---

## 💾 데이터 백업 / 초기화

**데이터 관리** 탭:
- **전체 백업(.db)** — 모든 데이터를 파일 하나로 내려받기. 복원은 그 파일을 데이터 폴더의 `attendance.db`에 덮어쓰기.
- **항목별 엑셀 저장** — 출석·주간·감정·포인트·도서·체육.
- **초기화(되돌릴 수 없음)** — 항목별(출석/감정/포인트/도서/체육) · 전체 활동기록(등록은 유지) · 공장 초기화(전부 삭제). 초기화 전 **백업 권장**.

## 📗 (선택) 구글 시트 동기화

출석을 구글 시트에 자동으로 올립니다. 켤 때만 동작합니다.

1. 구글 시트 → **확장 프로그램 → Apps Script** 에 `tools/google-apps-script.gs` 내용 붙여넣고 저장
2. **배포 → 새 배포 → 웹 앱**, 액세스 **"모든 사용자"** 로 배포 → **웹 앱 URL** 복사
3. 출석앱 **설정 → Google Sheets 동기화** 에 URL 붙여넣고 **연결 테스트 → 사용 ON**

→ 이후 태그할 때마다 `출석` 시트에 한 줄씩 추가, [지금 전체 동기화]로 `주간점수` 시트 갱신.

## 🌐 (선택) 공공데이터 연동 (무료 · data.go.kr)

인증키 1개로 두 기능을 켤 수 있습니다.

1. **data.go.kr** 가입 후 아래 서비스 **활용신청**(즉시 승인):
   - 한국천문연구원_특일 정보 (공휴일)
   - 한국환경공단 에어코리아_대기오염정보 (미세먼지)
   - 기상청_단기예보 조회서비스 (날씨)
2. 마이페이지의 **일반 인증키(Decoding)** 복사 → **설정 → 공공데이터 연동** 에 붙여넣기
3. 원하는 기능 ON:
   - **📅 공휴일 자동 반영** — 주말·공휴일을 주간 점수의 결석에서 제외
   - **🌤 미세먼지·날씨** — 실시간 출석/셔틀런/서킷 상단 위젯, 야외활동 가능/실내 권장 판단

> 끄면 외부 통신 0(완전 오프라인). 키가 없거나 조회 실패해도 출석 기능엔 영향 없습니다.

## 🏫 (선택) NEIS 학교 연동 (무료 · 급식·학사일정)

NEIS 교육정보 개방 포털 인증키 1개로 두 기능을 켤 수 있습니다.

1. **open.neis.go.kr** 가입 후 **인증키 신청**(즉시 발급)
2. **설정 → 🏫 NEIS 학교 연동** 에 인증키 붙여넣기
3. **학교명 검색 → 우리 학교 선택**(교육청·학교 코드 자동 입력)
4. 원하는 기능 ON:
   - **🍱 오늘 급식 표시** — 실시간 출석 화면에 메뉴·알레르기 번호 표시
   - **📆 학사일정 반영** — 휴업일·방학을 주간 점수의 결석에서 자동 제외(시험 등 행사는 주간표에 표시)

> 공휴일·날씨와 마찬가지로 켤 때만 동작하며, 실패해도 출석 기능엔 영향 없습니다.

## 🕵️ 리더기 규격(카드번호 형식) 확인 — 시리얼 리더기 처음 연결 시

리더기가 카드 태그 시 COM 포트로 보내는 데이터 형식을 확인합니다.

```
node tools/sniff.mjs COM9          # 카드를 한 번 대보세요 (HEX/TEXT 출력 확인)
node tools/sniff.mjs COM9 115200   # 글자가 깨지면 다른 속도로
```

확인한 포트·속도·형식을 `config.json`에 반영합니다.
(자동 탐지가 기본이며, 설정·진단 탭에서 포트를 직접 고를 수도 있습니다.)

> sniffer와 본 프로그램은 동시에 COM 포트를 열 수 없습니다. 확인 후 Ctrl+C로 닫고 실행하세요.

---

## ⚙ 설정 (`config.json`)

```jsonc
{
  "serial":     { "autoDetect": true, "port": "COM9", "baudRate": 9600 },
  "parser":     { "mode": "stx-etx", "trim": true, "delimiter": "\r\n" },
  "attendance": { "debounceSeconds": 5 },          // 같은 카드 5초내 재태그 무시
  "scoring":    { "fullScore": 10, "onTimeBy": "09:00" }, // 정시 마감·만점 (설정 탭에서도 변경)
  "server":     { "port": 3000 }
}
```

- 점수·구글 시트·공공데이터 등 화면에서 바꾼 설정은 데이터 폴더의 `settings.json`에 저장되어 다음 실행에도 유지됩니다.
