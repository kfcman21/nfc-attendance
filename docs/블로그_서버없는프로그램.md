# 🏷️ 서버 없이, 인터넷 없이 — NFC 카드 한 장으로 굴러가는 '우리 반 만능 프로그램' 만들기

> **PC 한 대 + 카드 리더기 한 대로 출석·감정·독서·체육까지 전부 기록한다고?**
> 클라우드도, 별도 서버도, 월 사용료도 없는 "서버 없는 프로그램" 개발기 🙌

---

## 📌 들어가며 — 왜 '서버 없는' 프로그램일까?

요즘 프로그램이라고 하면 으레 이렇게 생각하죠.

- 클라우드 서버를 빌리고 💸
- 데이터베이스 서버를 따로 두고 🗄️
- 인터넷이 끊기면 먹통 😱

그런데 학교 교실을 생각해 보면, 이런 게 다 **부담**입니다.
선생님이 IT 관리자가 아니잖아요. 인터넷이 느릴 수도 있고요.

그래서 이번엔 **완전히 반대로** 가봤습니다.

> **"서버도, 인터넷도, 설치도 필요 없이 — exe 파일 하나 더블클릭하면 끝!"**

NFC 카드 리더기(**CR-100**)를 USB로 꽂고, 학생들이 카드를 태그하면
출석부터 독서 기록, 체육 기록까지 전부 **PC 한 대 안에서** 처리됩니다.
인터넷 선을 뽑아도 그대로 돌아가요. 이게 바로 오늘의 주제, **서버 없는 프로그램**입니다. 😎

---

## 🔌 1단계 — 리더기랑 '말'부터 트기

가장 먼저 한 일은 **리더기가 무슨 말을 하는지 알아내기**였어요.

NFC 리더기는 USB로 꽂으면 'COM 포트'라는 통로로 데이터를 보냅니다.
근데 어떤 형식으로 보내는지는 제조사마다 달라요. 그래서 진단 도구를 만들어
카드를 한 번 태그해 봤더니…

```
[오후 5:42:09] 13 bytes
   HEX  : 02 36 38 30 30 36 39 34 38 43 35 0D 0A 03
   TEXT : ·68006948C5··
```

오! 해석해 보니 이런 규칙이었어요. 👇

- `02` = 시작 신호(STX)
- `68006948C5` = **카드 고유번호(UID)**
- `0D 0A` = 줄바꿈
- `03` = 끝 신호(ETX)

이 규칙만 알아내면, 카드를 댈 때마다 **누구 카드인지** 정확히 잡아낼 수 있습니다. 🎯

> 💡 **TIP** — 리더기는 '제품마다 말투가 다른 외국인'이라고 생각하면 쉬워요.
> 먼저 무슨 말을 하는지 받아 적는 게 1순위!

---

## 🧱 2단계 — '서버 없이' 만드는 비밀

원래 Python으로 만들려고 했는데, 테스트한 PC에 Python이 없더라고요.
대신 **Node.js**가 깔려 있어서 방향을 틀었습니다. 그런데 이게 신의 한 수였어요.

핵심 재료는 이렇습니다.

| 재료 | 역할 | 왜 좋은가 |
|---|---|---|
| **Electron** | 데스크톱 창(앱) | 더블클릭하면 창이 뜸 |
| **내장 SQLite** | 데이터 저장 | DB 서버 설치 불필요! |
| **serialport** | 리더기 통신 | 카드 데이터 읽기 |

여기서 포인트는 **데이터베이스 서버가 따로 없다**는 거예요.
보통 DB라고 하면 MySQL 같은 걸 설치해야 하는데, Node에 **SQLite가 내장**돼 있어서
파일 하나(`attendance.db`)에 모든 기록이 차곡차곡 쌓입니다.

> 그래서 "서버 없는 프로그램"이 가능해진 거죠.
> 클라우드 ❌ / DB 서버 ❌ / 인터넷 ❌ — **그냥 PC 한 대면 끝.**

---

## 🎨 3단계 — 기능이 점점 불어나다

처음엔 '출석'만 만들려고 했는데… 하다 보니 욕심이 생겨서 (ㅋㅋ)
교실에서 쓸 수 있는 기능이 계속 붙었습니다. 하나씩 소개할게요!

### ✅ ① 실시간 출석
카드를 대면 화면에 학생 이름이 **큼지막하게** 뜨면서 출석 완료!
같은 카드를 5초 안에 또 대면 중복 처리는 자동으로 막아줍니다.

### 😊 ② 감정 출석부 ('오늘의 기분')
출석하면 바로 **"오늘 기분이 어때요?"** 화면이 떠요.
😄🙂😐😫😢😡 중에 하나 누르면 끝. 아이들 마음 상태를 매일 기록할 수 있죠.

### 📊 ③ 대시보드 & 감정 통계
- 오늘 출석률, 최근 7일 추이를 **그래프**로
- 우리 반 **'감정 날씨'**(☀️🌤️⛅🌧️⛈️)를 한눈에
- **관심이 필요한 학생**(피곤·슬픔·화남 표시)을 따로 모아줘서 선생님이 챙기기 좋아요 🤗

### 🏆 ④ 칭찬 포인트 상점
카드 대고 +1, +5, +10 포인트! 학급 랭킹판도 자동으로 만들어집니다.

### 📖 ⑤ 도서 대여 & 학급 독서 온도계
- **[학생 카드] → [책 카드]** 순서로 태그하면 대여!
- 책만 다시 태그하거나 목록의 **[반납]** 버튼으로 반납
- 반납하면 **별점 ⭐ + 한 줄 평**을 남겨 '나만의 독서 기록장'이 쌓여요
- 반 전체가 읽은 책을 합산해 **'학급 독서 온도계'** 🌡️ 가 쭉쭉 올라갑니다

### 🏃 ⑥ 셔틀런 자동 기록기
반환점에 리더기를 두고 왕복할 때마다 카드 태그! 숫자가 실시간으로 올라가고,
**개인 최고 기록을 깨면 🎉 축하 효과**까지 터집니다. 페이스 가이드 비프음은 덤!

### 🏋️ ⑦ 서킷 트레이닝 타이머
운동 구역마다 카드를 붙여두고 **[학생]→[종목]** 으로 시작, 종목 카드로 종료.
수행 시간이 저장돼서 **'개인 체력 성장 그래프'** 를 볼 수 있어요.

### 🗑️ ⑧ 카드 초기화 & ⚙️ 설정·진단
카드를 다른 용도로 다시 쓰고 싶을 때 초기화하고,
리더기가 말썽이면 **진단 화면**에서 직접 포트를 고를 수 있게 했습니다. (이게 나중에 효자 노릇을 해요!)

---

## 🖤 4단계 — 디자인은 '나이키'처럼

기능이 많아지니 화면이 좀 산만해졌어요. 그래서 **나이키 디자인 시스템**을 참고했습니다.

- 순수 **검정 · 흰색 · 회색** + 포인트 색은 딱 2개(세일 레드 / 성공 그린)
- 그림자 없이 **각진 카드 + 얇은 선**
- 큰 숫자는 **Bebas Neue** 폰트로 '빌보드'처럼 시원하게
- 버튼은 전부 **검정 알약(pill)** 모양

여기에 **로고**도 직접 그리고, **라이트/다크 모드** 토글까지 넣으니
"오, 이거 진짜 제품 같은데?" 소리가 절로 나왔습니다. 🤩

---

## 📦 5단계 — exe 파일 하나로 배포!

자, 이제 진짜 핵심. **다른 PC에 어떻게 주지?**

`electron-builder`로 묶었더니 두 종류가 나왔어요.

- **설치본** (`NFC학생출석-설치.exe`) — 시작메뉴·바탕화면 바로가기 자동 생성
- **포터블** (`NFC학생출석-portable.exe`) — 설치 없이 더블클릭 실행

그리고 가장 자랑하고 싶은 부분 👇

> **별도 Node.js 설치조차 필요 없습니다!**
> Electron 안에 Node가 내장돼 있어서, 리더기 모듈까지 그대로 작동해요.
> 진짜 말 그대로 **"exe 하나만 복사하면 끝"**.

데이터는 `%AppData%` 폴더에 저장돼서 앱을 옮기거나 새로 깔아도 기록이 안전하게 보존됩니다. 👍

---

## 😅 6단계 — 진짜 복병: "노트북에서 카드 인식이 안 돼요!"

데스크톱에선 완벽했는데, **다른 노트북에 깔았더니 카드 반응이 없는** 거예요. 😱

처음엔 "포트가 잘못 잡혔나?" 싶어서
- 리더기를 **자동으로 찾는 기능**(USB 포트가 바뀌어도 연결)
- 화면에서 **포트를 직접 고르는 진단 탭**
- 들어오는 데이터를 **실시간으로 보여주는 기능**

까지 다 넣고 점검했는데… 포트는 정확히 맞았어요. 그런데 **데이터가 한 글자도 안 들어와요.** 🤔

범인은 의외였습니다. **노트북 USB 전력 부족!**

NFC 리더기는 카드를 읽을 때 전파를 쏘느라 전력이 꽤 필요한데,
노트북은(특히 배터리 모드일 때) USB 전력이 약해서 **리더기가 연결만 되고 카드를 못 읽던** 거였어요.

✅ **해결책: 노트북을 충전기에 꽂고 + 본체 USB 포트에 직접 연결!**

> 💡 **교훈** — "연결됨"이라고 다 되는 게 아니다.
> 데이터가 실제로 들어오는지 **눈으로 확인하는 진단 기능**이 결국 문제를 잡았습니다.

## 🎁 마치며 — 서버 없는 프로그램의 매력

이번 프로젝트로 확실히 느낀 점.

- ✨ **인터넷 없이도** 충분히 강력한 프로그램을 만들 수 있다
- ✨ **서버·DB 설치 없이** 파일 하나로 배포할 수 있다
- ✨ 교실 같은 현장엔 **'단순하고 튼튼한 게' 최고**다

출석 하나로 시작했는데, 어느새 **출석·감정·독서·체육·포인트**까지 담은
'우리 반 만능 프로그램'이 됐네요. 그것도 **서버 한 대 없이!** 🎉

혹시 비슷한 걸 만들고 싶으신 분께 한 줄 요약:

> **"Electron + 내장 SQLite = 서버 없는 데스크톱 앱의 정석!"**
