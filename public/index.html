                📗 Google Sheets 동기화

                출석을 구글 시트에 자동으로 올려 어디서든 보고 공유합니다. 완전 오프라인으로 쓰려면 꺼두면 되고, 켜져 있을 때만 동작합니다.

                ① 처음 한 번: 구글 시트 연결 방법 (클릭)
                1. 구글 시트 새로 만들기 → 메뉴 확장 프로그램 → Apps Script.
                2. 프로젝트 폴더의 tools/google-apps-script.gs 내용을 붙여넣고 저장.
                3. 배포 → 새 배포 → 웹 앱, 액세스 권한 "모든 사용자"로 배포.
                4. 나오는 웹 앱 URL(…/exec)을 아래에 붙여넣고 연결 테스트.

                🌐 공공데이터 연동 (무료 · data.go.kr)

                공휴일 자동 반영(주간 점수 정확도)과 미세먼지·날씨(체육 야외활동 판단)를 켭니다. 꺼두면 외부 통신 없이 동작합니다.

                ① 처음 한 번: 무료 인증키 발급 방법 (클릭)
                1. data.go.kr 회원가입 후, 아래 3개 서비스에서 활용신청(즉시 승인):
                2. · 한국천문연구원_특일 정보 (공휴일)
                3. · 한국환경공단 에어코리아_대기오염정보 (미세먼지)
                4. · 기상청_단기예보 ((구)동네예보) 조회서비스 (날씨)
                5. 마이페이지 → 일반 인증키(Decoding)를 복사해 아래에 붙여넣기.

                공휴일·미세먼지·날씨에 각각 다른 키를 쓰려면 아래에 개별로 입력하세요. (비우면 위 공통 키를 사용)

                📅 공휴일 자동 반영 (주말·공휴일을 결석에서 제외)
                🌤 미세먼지·날씨 (출석·체육 화면 위젯, 야외활동 판단)
                기상청 격자값 (서울=60,127 · 모르면 그대로)

                🏫 NEIS 학교 연동 (무료 · 급식·학사일정)

                오늘 급식 식단·알레르기를 실시간 출석 화면에 표시하고, 학사일정(휴업일·방학)을 주간 점수의 결석 계산에서 자동 제외합니다.

                ① 처음 한 번: NEIS 인증키 발급 (클릭)
                1. open.neis.go.kr(교육정보 개방 포털) 회원가입 → 인증키 신청(즉시 발급).
                2. 발급받은 인증키를 아래에 붙여넣기.
                3. 학교명을 검색해 우리 학교를 선택하면 끝(교육청·학교 코드 자동 입력).
                🍱 오늘 급식 표시 (실시간 출석 화면)
